## Installation

Installing in `~/.config/nvim/lua/custom`

```bash
cd ~
git clone https://github.com/axemanofic/nvchad-config .nvchad
ln -s "$HOME/.nvchad/custom" "$HOME/.config/nvim/lua/custom"
```
